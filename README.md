# ft_core
To create resources on FiveM

### Prerequisites

* [Nodejs](https://nodejs.org/)

### Installing

Copy `exemple` directory and rename to `ft_exemple`.

Last add, into the server.cfg file :
 ```
    start ft_exemple
```

### Documentation

- Online : https://ft_core.fivem.tools
- Local : `docs` directory (github)
